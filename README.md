# COMPUTER SHOP

The Integrated Management System for Computer and Maintenance Stores is a technological solution developed to optimize and organize the daily
operations of a store specializing in IT products and services. This system ranges from inventory control to sales management and technical 
assistance services, providing efficient and integrated management to serve customers, such as registering employees, customers, and products.

This project was developed for the software engineering course

this project support:
Product inventory control (hardware, software, peripherals, etc.);
Sales management (budgets, orders, invoicing);
Registration and monitoring of technical assistance service orders;
Customer registration and purchase history;
Basic financial control (sales and purchases of products).
